# Sisfo Keuangan & CRM - Speaking Partner 🚀

Sistem informasi keuangan terintegrasi dengan CRM Lead Management berperforma tinggi.

## 🚀 Fitur Baru: Student Portal & Role Transparency (v4.1) 🎓 (TERBARU!)

### 1. Student Access Portal (Self-Service) 📱
*   **Modern Glassmorphism UI**: Antarmuka dashboard siswa yang premium dengan efek glassmorphism.
*   **Active Class Monitor**: Murid dapat melihat daftar kelas, jadwal harian, dan tutor pengajar.
*   **Learning Performance Tracking**: Visualisasi nilai rata-rata dan persentase kehadiran real-time.
*   **Floating WhatsApp Support**: Bantuan langsung ke Customer Care via tombol melayang.

### 2. Enhanced Role Security & Stability 🔐
*   **Robust Role Detection**: Sistem API kini mendukung deteksi role ganda (`role` & `roleSlug`) untuk memastikan transisi login yang mulus bagi Admin dan Staff.
*   **Frontend Data Hardening**: Proteksi anti-crash pada modul admin untuk menjamin stabilitas antarmuka saat sinkronisasi data database.
*   **Global Notification System**: Integrasi `react-hot-toast` di seluruh dashboard untuk feedback aksi yang lebih intuitif.

### 3. Payroll & Performance Transparency 💰
*   **Real-time Payroll Estimate**: Staff, CS, dan Pengajar dapat melihat estimasi gaji bulan berjalan secara transparan.
*   **Dynamic Incentive Logic**: Perhitungan bonus closing dan honor mengajar otomatis.
